# Proyecto de Variables de Sesión con Express

Este es un proyecto educativo desarrollado en Node.js y Express para demostrar la creación, persistencia y destrucción de variables de sesión utilizando `express-session`.

## 📸 Vista previa del funcionamiento
![Demostración de la Sesión Activa](image_359bcb.png)

## Tecnologías utilizadas:
* Node.js
* Express.js
* express-session

## Cómo ejecutarlo:
1. Clonar el repositorio.
2. Ejecutar `npm install` para restaurar dependencias.
3. Iniciar con `node server.js`.
